# CRSearch

[cpprefjp](https://cpprefjp.github.io/) と [boostjp](https://boostjp.github.io/) をいい感じに検索するウィジェット

- [x] cpprefjp
- [ ] boostjp （対応予定）

## ビルド

```bash
git submodule update --init
npm install
npm run build
```

## 開発

```bash
git submodule update --init
npm install
npm run dev
```

http://localhost:8080/
